   OR
   bash QUICKSTART.sh (Mac/Linux)

3. Open browser to: http://localhost:8000

OPTION 2 - MANUAL:
1. Read: README.md for detailed setup
2. Configure SQL Server database (see DATABASE_SETUP.md)
3. Start backend: cd backend && python run.py
4. Start frontend: cd frontend && python -m http.server 8000
5. Open: http://localhost:8000
